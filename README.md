## Contexto de la aplicación

Esta plataforma funciona como un **dashboard centralizado de seguridad y calidad de código**, pensado para equipos de desarrollo, DevSecOps y ciberseguridad.

A partir de los análisis realizados por SonarQube, la aplicación:

- Presenta métricas clave de seguridad (vulnerabilidades, bugs, code smells)
- Permite explorar hallazgos detallados por severidad
- Facilita el seguimiento histórico de escaneos
- Centraliza múltiples proyectos en un solo panel

La interfaz está diseñada para ser clara, moderna y orientada a la toma de decisiones.



## Tecnologías utilizadas

- React (Frontend)
- JavaScript / TypeScript (según implementación)
- CSS / Tailwind / Material UI (según tu stack)
- API de SonarQube

## Instalación y ejecución

Sigue estos pasos para correr el proyecto en tu entorno local:

### 1. Clonar el repositorio

```bash
git clone https://github.com/tu-usuario/tu-repositorio.git
cd tu-repositorio
````

### 2. Instalar dependencias

```bash
npm install
```

### 3. Iniciar la aplicación

```bash
npm run dev
```

La app estará disponible en:

```
http://localhost:5173
```

## Objetivo del proyecto

Este proyecto busca cerrar la brecha entre:

* Datos técnicos complejos
* Necesidad de visibilidad clara para decisiones rápidas

Convirtiéndose en una herramienta útil para:

* Equipos de desarrollo
* Analistas de seguridad
* Ingenieros DevSecOps
